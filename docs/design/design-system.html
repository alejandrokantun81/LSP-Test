10 / INTEGRACIÓN

Integración con Tailwind CSS v4

Los tokens en globals.css se mapean automáticamente a utilidades Tailwind. El bloque @theme genera clases utility sin configuración adicional.

/* globals.css */ @import "tailwindcss"; /* Bloque @theme → genera utilidades automáticamente */ @theme inline { --color-lego-yellow: #FFD700; → bg-lego-yellow, text-lego-yellow --color-ink-800: #1C1C24; → bg-ink-800, text-ink-800 --font-display: 'Bricolage Grotesque'; → font-display --shadow-yellow-sm: 0 4px 12px...; → shadow-yellow-sm }

Regla de Oro del Amarillo
El amarillo LEGO es el alma de la marca. Presente siempre, nunca excesivo. Máximo 20% de cobertura visual. Úsalo en bordes de selección, estados activos, badges de categoría individual, y studs decorativos. Nunca como fondo de página completa.
